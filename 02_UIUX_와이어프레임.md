## 2-2. 사용자 흐름
```
[랜딩] → 구글 로그인
 → [신규] 캐릭터 생성(외형/직업/닉네임) → 튜토리얼 퀘스트
 → [기존] 캐릭터 선택 → 대시보드
 → 일정 등록(＋, 제목/날짜/반복/난이도) → "퀘스트 수락!"
 → 체크박스 클릭 → 완료 애니메이션 + EXP 연출
 → EXP 누적 → 게이지 충전 → 레벨업 풀스크린
 → 보상 수령(코인/아이템) → 상점·꾸미기 사용
```
온보딩 핵심: 첫 일정 등록 → 즉시 완료 → 레벨업까지 3분 안에 경험.

## 2-3. 디자인 시스템
- **폰트:** 한글 둥근모꼴/갈무리11, 영문·숫자 Press Start 2P / Galmuri. `image-rendering: pixelated`.
- **컬러:** 배경 `#F4E4C1`/`#2B2B3D`(다크), 포인트 `#FF8C42`, EXP골드 `#FFD23F`, 성공 `#6BCB77`, 경고 `#FF5252`, 테두리 `#3A2E2A`.
- **도트 이펙트:**
  - 완료: 항목 회색 페이드 + `+EXP` 숫자 튀어오름(0.6s ease-out)
  - 레벨업: 황금 플래시(0.2s) → 캐릭터 점프 + 8방향 파티클 → "LEVEL UP!" 배너 슬라이드 인
  - 코인: HUD 메소로 빨려드는 모션
  - 원칙: 픽셀 그리드 단위 이동(서브픽셀 금지).

## 2-4. 반응형 전략
- PC(≥1024): 3단 고정, 캘린더 풀사이즈
- 태블릿(768~1023): 2단, 친구 패널 토글
- 모바일(<768): 세로 스택, 주간 뷰 기본, 하단 탭바
- 도트 깨짐 방지: 정수 배율 @1x/@2x/@3x만, `image-rendering: pixelated` 전역, 8의 배수 그리드.
