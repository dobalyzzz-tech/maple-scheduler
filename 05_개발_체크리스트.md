# 05. 개발 및 검수 체크리스트

## 5-1. 기능 구현
- [ ] 구글 OAuth 로그인/로그아웃, 가입 시 profiles 자동 생성 트리거
- [ ] 일정 CRUD 전체 테스트
- [ ] 일회성 일정 등록·완료
- [ ] 반복 일정 RRULE 전개 정확도, schedule_logs 날짜별 분리
- [ ] 친구 신청·수락·삭제, 협동 퀘스트 기여 합산
- [ ] EXP 난이도 차등/레벨업 임계치·이월 검증(서버 RPC 단위 테스트)
- [ ] 코인·아이템 지급/차감 트랜잭션 무결성

## 5-2. 게이미피케이션 검수
- [ ] 레벨업 효과음+풀스크린 애니메이션, 중복 트리거 방지
- [ ] 완료 시 +EXP 모션·사운드 0.3s 이내 반응
- [ ] 커스터마이징 저장/불러오기, 새로고침 후 유지
- [ ] 사운드 토글, 자동재생 차단 준수
- [ ] 배지·칭호 조건 발동 및 알림

## 5-3. 보안·예외
- [ ] 미로그인 접근 차단(라우트 가드 + 서버 검증)
- [ ] RLS: 본인 데이터만 read/write, 친구는 accepted만 제한 read
- [ ] EXP/코인은 서버 RPC로만 변경(클라 차단)
- [ ] 잘못된 입력 예외(빈 제목/과거 날짜/잘못된 RRULE)
- [ ] XSS sanitize, rate limiting

## 5-4. 테스트 케이스
- [ ] 렌더링 부하: 100/300/500개, 100개 초기렌더 <1s, 가상화 전후 비교
- [ ] EXP 정합성: SUM(schedule_logs.exp_gained) == profiles.exp
- [ ] 동시성: 더블클릭 시 EXP 1회만(idempotency)
- [ ] 반복 1년치 전개 성능/메모리
- [ ] 네트워크 예외 롤백·재시도
- [ ] 크로스 디바이스 Realtime 동기화
